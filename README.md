# Math Aggregator

This repository hosts the configuration and scripts required to build the Math Aggregator.

The [build script](.github/workflows/build.yml) is run by GitHub Actions to fetch the feeds every two hours, and the built website itself is hosted on GitHub Pages. All of the infrastructure needed to build/host is provided by GitHub.

## List of Feeds

Pull requests to update the list of feeds are welcome. Please only make changes to [math-aggregator.ini](math-aggregator.ini).
